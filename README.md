## Чтение и отправка сообщений в чат с помощью Asyncio Streams

## Установка
Скачайте репозиторий с кодом, установите необходимые зависимости:
```
pip install -r requirements.txt
```

## Запуск

Если вы впервые пользуетесь чатом, для начала понадобится регистрация. Выполните команду в терминале:
```
python registration.py
```
Чтобы открыть клиент чата выполните команду:
```
python client.py
```

### Настройка
Вы можете указать некоторые параметры для настрйки работы приложения:

- host_name сервера с чатом:
  + `HOST` для *.env*
  + `--host` для cli
- порт сервера для получения сообщений:
  + `WRITING_PORT` для *.env*
  + `--writing_port` для cli
- порт сервера для отправки сообщений:
  + `READING_PORT` для *.env*
  + `--reading_port` для cli
- Путь к файлу, в котором будет сохраняться история переписки: 
  + `CHAT_FILE_PATH` для *.env*
  + `--chat_file_path` для cli
- Ваш hash_id, полученный при регистрации (актуален для отправки сообщения):
  + `CHAT_HASH_ID` для *.env*
  + `--chat_hash_id` для cli
